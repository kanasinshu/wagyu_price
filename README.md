# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

* リモートリポジトリとは、githubなどネットワーク上でファイルの変更履歴を管理をする場所。ローカルリポジトリとは、作業のため作業者のPC上でファイルの変更履歴を管理をする場所。

## プッシュとマージの違いは何でしょうか？

* プッシュとは作業したファイルをリモートリポジトリへアップロードすること。マージとはマスター（メイン）ブランチに他のブランチの作業を取り込むこと。

## コミットとプッシュの違い

* コミットは作業履歴をローカルリポジトリに記述することで、プッシュはローカルリポジトリ内の任意のファイルの同期

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

* 作業内容を簡潔に表した文章を記述することが好ましい。

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

* ローカルでマージをする場合、不具合が起きることを予見できない可能性がある。プルリクエストでマージする場合はコードレビューをしてもらうため不具合が起きることを軽減することができる。

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

* コンフリクトを起こした箇所の作業者と相談し、どの変更を取り込むか相談をして対処をする。
